# Node.js - Environment Setup

## The Node.js Runtime
소스 파일에 쓰여진 소스 코드는 단순히 javascript이다. Node.js 인터프리터는 자바스크립트 코드를 해석하고 실행하는 데 사용될 것이다.

## Download Node.js
* 다운로드 링크: https://nodejs.org/en/download/  
```bash
export PATH=$PATH:/usr/local/nodejs/bin
```

```bash
# 코드 실행
node filename.js
```
